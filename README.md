📘 MoroHub AI – README

🚀 Overview
MoroHub AI is an interactive, voice‑enabled support assistant built using modern web technologies. It provides real‑time speech recognition, natural language responses, and voice output to deliver a smooth conversational experience.

This project was originally scaffolded using packages, then customized and branded for MoroHub.

🧠 Features
🎤 Voice Input (STT) – Real‑time speech‑to‑text using ElevenLabs

🗣 Voice Output (TTS) – Natural voice responses

🤖 AI Assistant – Conversational logic powered by your backend/LLM

⚡ Fast Frontend – Built with React + Vite

🌐 Deployed on Cloudflare Pages

🎨 Custom MoroHub Branding (favicon, metadata, OG tags)


🛠 Tech Stack
Layer	Technology
Frontend	React, TypeScript, Vite
UI	ShadCN UI, TailwindCSS
State/Data	React Query
Hosting	Cloudflare Pages
Voice	ElevenLabs STT + TTS
Routing	React Router
